# 📅 סדר יום דיגיטלי (Daily Schedule)

מערכת לניהול סדר יום ויזואלי ואינטראקטיבי, המותאמת לתלמידים עם אוטיזם. המערכת נועדה לקדם עצמאות, לספק משוב חיובי מידי ולמנוע הסחות דעת.

## 🎯 חזון הפרויקט (Vision)

המטרה היא ליצור סביבה טכנולוגית "נקייה" המאפשרת לתלמיד לנהל את סדר יומו בביטחון ובעצמאות.

- **עצמאות**: צמצום התלות במבוגרים לצורך מעבר בין פעילויות.
- **מכשיר נקי (Clean Device)**: המערכת רצה על מכשיר ייעודי (Kiosk Mode) ללא גישה ליוטיוב או אפליקציות מסיחות דעת אחרות, כדי למנוע "לופים" וחזרתיות ולהשאיר את הפוקוס על התקשורת והעשייה.
- **משוב חיובי**: חיזוקים ויזואליים וקוליים מידיים בעת הצלחה.

## ✨ תכונות מרכזיות (Features)

### 📋 לוח סדר יום (Current Status)

- **וויזואליות**: כרטיסיות גדולות וברורות עם תמונות/סמלים לכל פעילות.
- **אינטראקטיביות**: סימון סיום משימה בלחיצה או גרירה.
- **משוב רנדומלי**: בעת סיום משימה, המערכת משמיעה חיזוק קולי ("אתה אלוף!", "כל הכבוד!") מתוך בנק משפטים רנדומלי, כדי למנוע קיבעון על צליל ספציפי.
- **מצב עריכה (Admin Mode)**: ממשק נעול לשינויים ע"י הילד, עם אפשרות להורים/מורים להוסיף, לערוך ולסדר משימות.

### 🚀 בתכנון (Roadmap)

- **דאשבורד כיתתי/משפחתי**: מסך גדול (TV) שישקף בזמן אמת באיזו פעילות נמצא כל תלמיד/ילד, לצורך בקרה שקטה ללא צורך בשאלות.
- **תקשורת מהירה**: שילוב כפתורי תקשורת בסיסיים ("סיימתי", "צריך עזרה") בתוך הממשק.

## 🛠️ טכנולוגיה (Tech Stack)

הפרויקט נבנה בטכנולוגיות ווב מודרניות כדי להבטיח ביצועים מהירים, אנימציות חלקות ותחזוקה קלה:

- **Framework**: [SvelteKit](https://kit.svelte.dev/) עם **Svelte 5** (שימוש ב-Runes לניהול State).
- **Deployment**: מותאם לרוץ על [Cloudflare Pages](https://pages.cloudflare.com/) (Edge Runtime).
- **Storage**: שמירת נתונים מקומית (LocalStorage) לפרטיות ומהירות, עם תשתית מוכנה לסינכרון ענן עתידי.
- **Styling**: CSS מודרני (כולל אנימציות וגרידיאנטים) למראה מזמין ונגיש.

## 💻 הרצה מקומית (Development)

```bash
# התקנת תלויות
npm install

# הרצת שרת פיתוח (משתמש ב-adapter-auto למניעת קריסות Miniflare מקומיות)
npm run dev

# בדיקת סוגים (Type Check)
npm run check
```

## 📦 פריסה (Deployment)

הפרויקט מוגדר לרוץ על Cloudflare Pages.
בבנייה לייצור (Production), המערכת משתמשת אוטומטית ב-`adapter-cloudflare`.

```bash
npm run deploy
```
